## Run Locally

Clone the project

```bash
  git clone https://github.com/1vivekanand/chat-app.git
```

Go to the project directory

```bash
  cd chat-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
